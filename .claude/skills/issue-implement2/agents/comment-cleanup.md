# コメント整理 — サブエージェント詳細手順

`// 動作:` / `// エラー:` 形式の動作コメントを、実装完了後のコードに適したドキュメントコメントに変換する。

---

## 対象

`// 動作:` / `// エラー:` 形式のコメントを持つ関数・メソッド。`grep -rn "// 動作:\|// エラー:" --include="*.go" .` で検出する。

## 変換先の形式

```
// {サマリ: 何をするか 1-2文}
//
//   - {エッジケース、エラー、制約、戻り値の要点}
```

## ルール

- コードから自明な説明は削除する
- 非自明な制約、フォールバック、エラー条件は残す
- 実装手順の説明は削除する

## 検証

変換後に `just fmt` を実行して整形を確認する。
